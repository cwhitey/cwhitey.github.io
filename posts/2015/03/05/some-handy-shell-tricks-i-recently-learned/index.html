Last week I discovered a couple of really handy shell tricks which I now use.

Double BANG

This first one I got off another developer I was pairing with at work. !! <ENTER> can be used to reference the previous command which was executed. It can also be used with other commands e.g. cat $(!!). So far the only real use for it I've found is if you've typed a command and then realised you needed to run it with sudo. For this you'd use sudo !! <ENTER>.

Double ESC

This brings us to the other, more useful, shortcut I learned which I found by accident when trying to ESC something. <ESC> <ESC> will give you the last command you typed, just with sudo prepended onto it. If you've already started typing a command and then realise you need sudo at the start, <ESC> <ESC> will add that for you and allow you to continue writing your command. Ever since I discovered this shortcut I have literally used it every day.

It's worth noting that I'm using oh-my-zsh with quite a few plugins, and I could only get the !! shortcut to work using Bash.
