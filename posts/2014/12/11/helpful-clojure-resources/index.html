"Controlling complexity is the essence of computer programming."

- Brian Kernigan

During my placement at Australia Post I spent a lot of time using Clojure. Clojure is a functional language on the JVM with LISP-like syntax. I've really enjoyed learning and using Clojure and, although I'm still learning, I've used some resources along the way which I think are great for Clojure beginners.

Setup

  • Light Table - A good IDE for learning Clojure.
  • Emacs - Alternatively, you can start off using Emacs. It has a higher learning curve but is highly configurable and works really well with Lisp-based languages.
  • Leiningen - Tool for easy Clojure project and dependency management (mandatory).

Tutorials

  • Learn X in Y Minutes - Quickest way to get a brief understanding of Clojure syntax.
  • puredanger's Tutorial - Written by an active Clojure contributer.
  • Brave Clojure - Very extensive but sometimes a little hard to follow.
  • Fast Track Clojure
  • Destructuring Tutorial/Reference

Challenges

  • 4clojure
  • Clojure Koans
  • Exercism - Exercises for 19 different languages, including Clojure.
  • Project Euler - Language agnostic, but these challenges taught me a lot about how to solve algorithms in a functional way vs imperative.

Books

  • Programming Clojure - Pragmatic Programmers
  • Clojure Programming - O'Reilly
  • Clojure Cookbook - O'Reilly

Web Development

  • Clojure Web Dev - A great introduction to web development in Clojure. Contains a very informative talk about building a backend supported web application, and includes code snippets and resources for further study.
  • What Web Framework Should I Use?
  • Luminus - A great Leiningen template for setting up skeletons of Clojure webapps with various component configurations. It's accompanied by some very informative docs which make learning web development in Clojure much easier.

Documentation & Tools

  • tryclj - An online REPL.
  • grimoire - One of the best online references for Clojure documentation.
  • Clojure Toolbox - An organised collection of useful libraries and tools for Clojure.
  • ClojureWerks - Another collection of quality Clojure libraries and their documentation.
  • Clojure Sphere - A listing of statistics for components in the Clojure ecosystem.
  • Clojure Docs - Community-powered clojure documentation.
  • lein-bikeshed - A Leiningen plugin which scans your project for bad code.
