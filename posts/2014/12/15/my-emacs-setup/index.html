I've been learning how to use Emacs for several months for Clojure development. I thought I would document my initial setup here and continue updating it as I find new packages and develop more useful/convenient keybindings. You can also find my up to date .emacs.d in my dotfiles GitHub repo.

I tried various starter kits such as Prelude, Emacs Starter Kit and Emacs Live. I eventually settled with Emacs Live for a couple of months. After getting frustrated while trying to (unsuccessfully) modify the Emacs Live setup, I started looking for other options. I then stumbled on oh-my-emacs. Oh-my-emacs provides you with a completely documented starting setup which benefits both beginners and experts of Emacs. It's configuration is contained within .org files which contains Emacs Lisp code within #+BEGIN_SRC and #+END_SRC tags. The .org files get parsed into .el files during Emacs initialisation, and you can prevent snippets of Emacs Lisp code from being parsed by changing #+BEGIN_SRC emacs-lisp to #+BEGIN_SRC emacs-lisp :tangle no.

Installation:

  • Mac: Emacs Mac port
  • Linux: Your Linux distro's package manager will likely contain Emacs.

Install oh-my-emacs by following these Quick Start steps.
